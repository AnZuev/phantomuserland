We look for programmers to join us

Namely, we need someone who will be working with:

  • Bytecode interpreter (and we need JIT too)
  • Graphics engine, font engine
  • Network stack
  • Drivers
  • ARM port
  • MIPS port
  • 64 bit verison (either Intel, ARm or MIPS)
  • Unit tests
  • Framebuffer video drivers
  • Java classlib
  • Unix compatibility subsystem
